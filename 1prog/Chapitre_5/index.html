Les structures

On crée un nouveau type, qui va regrouper plusieurs variables afin de les manipuler comme une seule.

Examples

Une structure "Point"


struct Point {
    double x, y;
};

...

Point a;
a.x = 2.3;
a.y = 3.4;

Une structure "Cercle"


struct Cercle {
    Point centre;
    double rayon;
    Color couleur;
};

...

Cercle C;
C.centre.x = 12.;
C.centre.y = 12.;
C.rayon = 10.4;
C.couleur = Red;

Utilisation


struct Point {
    double x, y;
};

...

Point a;
a.x = 2.3;
a.y = 3.4;

Point b = {6.3, 5.8}; // Fonctionne

a = b; // Fonctionne

a = {2.3, 3.4}; // Ne fonctionne pas
