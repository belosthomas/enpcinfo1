Installation de "Qt Creator - Version ENPC"

Procurez-vous "Qt Creator - Version ENPC"

  • Depuis une clé USB
  • Depuis la page dédié
Exécutez le fichier d'installation
Si vous voyez cette fenêtre, appuyez sur "Informations complémentaires"...
...Puis sur "Exécutez quand même"
Appuyez sur "Suivant"
Appuyez sur "Installer". Et acceptez les droit administrateur.
L'installation peut prendre beaucoup de temps. Profitez-en pour lire le cours !
Une fois l'installation fini, appuyez sur "Terminer", cela ouvrira Qt Creator.
Nous allons maintenant ouvrir un projet de test pour vérifier que tout fonctionne correctement.
Téléchargez le "Programme supplémentaire" du TP 1, sur le site de Pascal Monasse
Extrayez le fichier téléchargé en faisant un clic droit sur le fichier, et en appuyant sur "Extraire tout".
Déplacez le dossier extrait "Tp1_Final" dans le dossier C:/ENPC/Projects. C'est ici que vous créerez tout vos projet !
Ouvrez la fenêtre de Qt Creator. Appuyez sur le menu en haut à gauche "Fichier", puis sur "Ouvrir un fichier ou un projet...". Naviguez dans C:/ENPC/Projects/Tp1_Final et sélectionner le fichier "CMakeLists.txt".
Appuyez sur le bouton "Configure Project". Vous aurez à faire cette étape pour chacun de vos projets.
Appuyez sur le bouton vert en forme de triangle en bas à droite pour exécuter le projet. Si tout se passe bien, vous devriez voir une fenêtre apparaitre avec une balle rebondir à l'intérieur !
